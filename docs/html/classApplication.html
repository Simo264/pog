Descrizione dettagliata

La classe Application estende JFrame, rappresenta quindi la finestra principale in cui si aggiungeranno i vari componenti quali MenuBarComponent e il TablePanel.

Dato il file di configurazione "application.config" che si trova nella cartella configs/, vengono impostate le proprietà del JFrame (es. titolo, dimensione, posizione...)
