Detailed Description

ApplicationLogger estende la classe ApplicationFileWrapper, rappresenta il logger dell'applicazione (log/log.txt): vengono segnate tutte le informazioni riguardanti gli eventi dell'utente (apertura/salvataggio di un nuovo workspace, aggiornamento tabella...), informazioni su crash del programma e informazioni generali.
