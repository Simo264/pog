Descrizione dettagliata

La classe Workspace rappresenta il file di lavoro in cui si sta lavorando. All'avvio del programma il workspace di default sarà il file (se esiste) "workspace.config" all'interno della cartella configs/, altrimenti null.

Durante l'esecuzione del programma l'utente potrà decidere se aprire un nuovo spazio di lavoro e il workspace sarà il nuovo file creato.
